## How to Apply a Color Scheme

1. Open `styles.css` in a text editor
2. Find the `:root {` section at the top (around line 7)
3. Locate the color variables (lines 8-18)
4. Copy and paste your chosen color scheme
5. Save the file
6. Refresh your browser to see the changes

## Creating Your Own Color Scheme

### Tips for Choosing Colors:

1. **Primary Color** - Your main brand color
   - This will be most visible (buttons, headings, icons)
   - Should reflect your church's personality
   - Test for readability with white text

2. **Secondary Color** - Complementary accent
   - Should work well with primary color
   - Used for highlights and secondary elements
   - Provides visual interest

3. **Accent Color** - Additional emphasis
   - Optional third color for variety
   - Use sparingly for special elements
   - Should not clash with primary/secondary

### Color Psychology for Churches:

- **Blue** - Trust, peace, stability, heaven
- **Green** - Growth, life, renewal, nature
- **Purple** - Royalty, spirituality, dignity
- **Red** - Passion, sacrifice, love, energy
- **Gold/Yellow** - Glory, light, warmth, joy
- **Brown** - Earthiness, stability, foundation
- **Gray** - Wisdom, maturity, balance

### Tools for Creating Color Schemes:

- **Coolors.co** - Generate and customize palettes
- **Adobe Color** - Create color harmonies
- **Paletton** - Color scheme designer
- **ColorSpace** - Generate nice color palettes

### Accessibility Considerations:

1. **Contrast Ratio** - Ensure text is readable
   - Minimum 4.5:1 for normal text
   - Minimum 3:1 for large text
   - Test at: https://webaim.org/resources/contrastchecker/

2. **Color Blindness** - Avoid relying solely on color
   - Don't use red/green for important distinctions
   - Test your site at: https://www.color-blindness.com/coblis-color-blindness-simulator/

3. **Dark Variants** - Each color needs dark/light versions
   - Dark: 20-30% darker than base color
   - Light: 20-30% lighter than base color

## Testing Your Colors

Before finalizing:

1. **View on Different Screens**
   - Desktop monitor
   - Laptop
   - Tablet
   - Phone

2. **Check in Different Lighting**
   - Bright room
   - Dim room
   - Outdoor sunlight

3. **Get Feedback**
   - Show to church leadership
   - Ask congregation members
   - Consider all age groups

4. **Live Preview**
   - Use browser developer tools
   - Edit CSS live to test
   - Take screenshots to compare

## Common Color Mistakes to Avoid

❌ **Too Many Colors** - Stick to 2-3 main colors
❌ **Low Contrast** - Make sure text is readable
❌ **Trendy Over Timeless** - Choose colors that will age well
❌ **Ignoring Brand** - Match existing church materials
❌ **All Bright Colors** - Balance bright with neutral
❌ **No Variation** - Use light/dark variants of colors

## Matching Existing Church Materials

If your church already has:
- **Logo** - Use its primary colors
- **Building Colors** - Consider exterior/interior paint
- **Printed Materials** - Match bulletins, signs
- **Denominational Standards** - Follow guidelines if applicable

## Quick Color Updates

### Just Want to Change One Color?

**Change Primary Color Only:**
```css
--primary-color: #YOUR-COLOR;
--primary-dark: #DARKER-VERSION;
--primary-light: #LIGHTER-VERSION;
```

**To Get Darker/Lighter Versions:**
- Use: https://www.color-hex.com/
- Enter your color
- Click "Shades & Tints"
- Choose appropriate lighter/darker shades
