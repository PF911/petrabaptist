How to build the Ministries and Leadership submission forms

How It All Works Together

📝 Adding New Content

  • Use the Google Form
  • Fill out the fields
  • Submit — done
  • Site updates in ~10 min

✏️ Editing Existing Content

  • Open the Google Sheet
  • Click the cell to change
  • Type the new value
  • Site updates in ~10 min
Important: Forms are for adding new entries only. To change a meeting time, update a bio, or fix a typo on an existing entry — go directly to the sheet and edit that cell. Do not submit a duplicate form entry.

Build the Ministries Form

Create this form at forms.google.com. Title it "Add a Ministry — Petra Baptist Church".

Question Label (exact) Type Required? Notes / Help Text to Add
icon Short answer Yes Font Awesome icon class — see icon list below. e.g. fa-user-tie
title Short answer Yes Ministry display name, e.g. Petra Men
description Paragraph Yes The "About" paragraph shown in the popup
time Short answer Yes When the ministry meets, e.g. Last Saturday at 8:00AM
leader Short answer No Single leader name. Leave blank if using "leaders" field instead.
leaders Short answer No Multiple leaders, e.g. Brian & Cheri Lee. Use either this OR "leader" — not both.
externalLink Short answer No Full website URL if the ministry has its own site, e.g. https://petrawomen.net/
externalLinkText Short answer No Button label for the link above, e.g. Visit PetraWomen.net
photo Short answer No Full image URL. Must be 800×533px. Leave blank if no photo.
photo2 Short answer No Additional photo URL. 800×533px. Leave blank if not needed.
photo3 Short answer No Additional photo URL. 800×533px. Leave blank if not needed.
photo4 Short answer No Additional photo URL. 800×533px. Leave blank if not needed.
photo5 Short answer No Additional photo URL. 800×533px. Leave blank if not needed.
Linking to the existing sheet: After creating the form, go to the Responses tab → click the Google Sheets icon → choose "Select existing spreadsheet" → select the Petra Ministries sheet. Form submissions will add rows directly to that sheet automatically.

Create this form at forms.google.com. Title it "Add a Leadership Member — Petra Baptist Church".

Important Rules to Follow

  • !
    Question labels must match exactly. The question text in the form (e.g. icon, title, bio1) must be typed exactly as shown — lowercase, no spaces. These labels become the column headers in the sheet, which is how the website finds the data.
  • !
    Do not add a "Timestamp" column to the sheet manually. Google Forms adds this automatically. Leave it alone — the website ignores it.
  • !
    Photo dimensions are fixed. Ministry photos must be 800×533px. Leadership headshots must be 400×500px. No exceptions — wrong sizes will display incorrectly.
  • !
    Changes take 5–10 minutes to appear on the site due to Google's publishing cache. This is normal — not a bug.
  • !
    To edit existing content, use the sheet — not the form. Submitting the form again creates a duplicate entry. Open the sheet, find the row, click the cell, and type the correction.
  • !
    To remove a ministry or leader, delete their entire row from the sheet. Do not just clear the cells — delete the whole row so there is no blank entry.
